To compile/run C code:
	- Open Command Prompt (Not Git Bash)
	- Go to directory
	- "gcc -o any_name file_name.c" to compile
	- "./any_name" to run code

Alternatively (more recently):
	- Open Windows PowerShell
		- when going into directories with spaces use quotes:
			$ cd "Spring 2018"
	- Go to directory
	- "gcc -g -Wall -Wextra -Wpedantic -std=c99 -c -o any_name file_name.c" to compile with warnings
